## 3. 벤치마크 성능 분석

### CPU 성능 (Cinebench R23 기준)
M4 Pro는 이미 현재 세대를 선도하는 성능을 보여줍니다. M5 Pro는 더 많은 고성능 코어와 개선된 아키텍처 덕분에 약 20~30%의 추가적인 싱글/멀티 스레드 성능 향상이 예상됩니다.

- Single-Core: M4 Pro (1,850pts) vs M5 Pro (예상 2,300pts+)
- Multi-Core: M4 Pro (14,500pts) vs M5 Pro (예상 19,000pts+)

### GPU 성능 및 그래픽 처리
M5 Pro는 게임 및 3D 렌더링 작업에 있어 압도적인 차이를 보여줄 것입니다. 특히 Apple Silicon의 Metal API와의 최적화 덕분에 Ray Tracing 성능이 비약적으로 상승할 것으로 기대됩니다.

- GFXBench: M4 Pro 대비 약 40~50% 향상 예상
- RenderTime (Blender): 작업 시간 단축 효과 극대화
